---
title: Create a New Project
description: Start managing docs by creating a project in Lunda.
---

## Steps

<Steps>
  <Step title="Navigate to dashboard">
    Click "Dashboard" from the Lunda sidebar.
  </Step>
  <Step title="Start new project">
    Press "New Project"
    Enter a name and description.
  </Step>
  <Step title="Add documentation">
    Click "Add Document" inside the new project.
    Begin writing or importing files.
  </Step>
</Steps>

<Callout kind="info">
You can organize docs with folders after creating a project.
</Callout>
