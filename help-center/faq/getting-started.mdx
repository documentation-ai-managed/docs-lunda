## Overview

Lunda makes it easy to organize all your project documentation.

## Steps to Begin

1. Sign up at `https://lunda.io`
2. Create your first workspace
3. Add your first project
4. Invite team members (optional)

## Access Docs

- Use the left menu to browse documentation
- Click “New Document” to start writing

<Callout kind="info">
Use the search bar to quickly find topics.
</Callout>
