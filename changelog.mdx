## Upgrading Lunda

Follow these steps to update your Lunda instance safely.

<Steps>
  <Step title="Backup Configuration" icon="database">

    Export your project settings and docs from the dashboard.

    ```bash
    lunda export --project=my-docs --output=backup.json
    ```

  </Step>
  <Step title="Update Package" icon="download">
    
    Use your preferred package manager.

    <CodeGroup tabs="npm,yarn,pnpm">
      ````bash
      npm install lunda@latest
      ````
      ````bash
      yarn add lunda@latest
      ````
      ````bash
      pnpm add lunda@latest
      ````
    </CodeGroup>

  </Step>
  <Step title="Restart and Verify" icon="check-circle">

    Restart your Lunda server and test a page build.

    ```bash
    lunda start --verify
    ```

  </Step>
</Steps>

<Callout kind="tip">
  Review migration guides for breaking changes in each release above before upgrading production sites.
</Callout>
