## Advanced Editing and Formatting Options

You access a full-featured editor in Lunda with support for Markdown, rich text, and custom components. Real-time previews let you see changes instantly, while syntax highlighting ensures code blocks render perfectly.

<CodeGroup tabs="Markdown,Rich Text">
  ```markdown
  # Welcome to Lunda

  ## Key Features

  - **Bold** and *italic* text
  - `Inline code` for snippets
  - ```javascript
    console.log('Hello, Lunda!');
    ```

  > Blockquote for tips.
  ```
  ```html
  <h1>Welcome to Lunda</h1>

  <h2>Key Features</h2>
  <ul>
    <li><strong>Bold</strong> and <em>italic</em> text</li>
    <li><code>Inline code</code> for snippets</li>
  </ul>
  ```
</CodeGroup>

<Callout kind="tip">
  Use the `{/* Comment */}` syntax for MDX comments that don't appear in previews.
</Callout>

## Version History and Change Tracking

Lunda automatically saves every edit, creating a complete version history. You review changes, compare versions, and restore previous states with simple clicks.

<Steps>
  <Step title="View History" icon="clock">
    Open any document and click the history icon in the toolbar.
  </Step>
  <Step title="Compare Changes" icon="git-compare">
    Select two versions to see a side-by-side diff.
  </Step>
  <Step title="Restore Version" icon="refresh-cw">
    Click restore to revert the document to that state.
  </Step>
</Steps>

## Search and Filtering Documentation

Powerful search helps you locate content across your entire documentation space. Filter by tags, authors, or dates for precise results.

<Tabs>
  <Tab title="Basic Search" icon="search">
    Enter keywords in the global search bar to find matches instantly.
  </Tab>
  <Tab title="Advanced Filters" icon="filter">
    Combine tags like `feature` or `api` with date ranges: `created:>2024-01-01`.
  </Tab>
  <Tab title="Saved Searches" icon="bookmark">
    Save frequent queries for quick access from the sidebar.
  </Tab>
</Tabs>

Export your documentation in multiple formats or generate shareable links. You control access with passwords or public/private settings.
